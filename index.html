Learn Basic Database

SQL Server Management Studio (SSMS) adalah aplikasi perangkat lunak yang pertama kali diluncurkan dengan Microssoft SQL Server 2005 yang digunakan untuk mengonfigurasi, mengelola, dan mengelola semua komponen dalam Microsoft SQL Server.

SSMS sendiri adalah salah satu alat manajemen SQL Server yang digunakan untuk merancang query dan mengelola database dan Gudang data melalui computer pribadi atau cloud. SSMS menyediakan alat untuk mengonfigurasi, memantau , dan mengelola instance dan database SQL Server.
